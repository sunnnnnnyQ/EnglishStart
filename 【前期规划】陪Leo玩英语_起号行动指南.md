## 一、用旧号还是注册新号？

### 1.1 结论：建议用新号 ✅

| 方案 | 推荐度 | 理由 |
|------|:------:|------|
| **注册新号** | ⭐⭐⭐⭐⭐ | **强烈推荐** |
| 用旧号改造 | ⭐⭐ | 不推荐 |

### 1.2 为什么不建议用旧号？

**小红书的算法会给你的账号打"标签"**。你现在的旧号，算法已经根据你过去刷的内容、点赞、收藏行为，给你贴了一堆标签（比如旅游、美食、穿搭等等）。

如果你用旧号发英语启蒙的内容：
- ❌ 算法会困惑："这个号到底是干嘛的？"
- ❌ 你之前的粉丝（如果有的话）对英语启蒙不感兴趣，会拉低互动率
- ❌ 旧号可能有历史笔记/收藏，和新定位不匹配
- ❌ 账号权重标签混乱，新内容很难被精准推荐给目标人群

**新号的优势：**
- ✅ 算法从零开始认识你——你发什么内容，算法就给你打什么标签
- ✅ 新号有"冷启动流量扶持"——小红书会给新号一定的初始曝光机会
- ✅ 主页干净整洁，新粉丝进来一眼就知道你做什么
- ✅ 不会被旧内容干扰

### 1.3 新号注册注意事项

1. **用另一个手机号注册**（或者用现有号绑定的其他方式）
2. **注册后先不要急着发内容**，先完成账号装修（见下文）
3. **注册后先"养号"3-5天**：
   - 每天刷20-30分钟小红书
   - **只搜索和浏览英语启蒙相关内容**
   - 点赞、收藏一些英语启蒙的笔记
   - 关注5-10个英语启蒙类账号
   - **目的：让算法快速识别你的账号属于"英语启蒙"领域**

> 💡 **你的旧号可以继续用来刷小红书**，不用丢掉。新号专门用来做「陪Leo玩英语」。

### 1.4 养号期间推荐搜索/浏览的关键词

在新号上搜索以下关键词，浏览并点赞收藏相关优质内容：

```
· 英语启蒙
· 英语启蒙游戏
· 3岁英语启蒙
· 亲子英语
· 字母教学
· 绘本阅读方法
· 在家英语启蒙
· 磨耳朵
· 英语启蒙方法
· TPR教学
```

每天花10-15分钟即可，连续3-5天。

```

"A warm and cute cartoon illustration of a father and 
a 3-year-old boy, minimalist style, the boy is holding 
an alphabet block, warm yellow and sky blue color scheme, 
round avatar format, white background, kawaii style"
```

**方案B：实物照片**
- 拍一张爸爸和Leo的手部合照（拿着字母卡片）
- 或者拍Leo的背影+英语绘本/字母玩具
- 暖色调滤镜处理

**方案C：设计文字logo**
- 用你的设计能力做一个简洁的文字logo
- "陪Leo玩英语"+ 简单图形元素

> 💡 **我推荐方案A**，因为卡通形象辨识度最高，而且和"不出镜"的策略一致。你作为设计师，用AI出图+自己微调，应该很快就能做出一个好看的头像。

## 五、本周六素材拍摄清单

### 5.1 拍摄前准备

**提前准备好的物品：**
- [ ] 字母卡片（如果家里有的话；没有就用A4纸手写几个字母）
- [ ] 白纸/彩色纸 + 彩笔/马克笔
- [ ] Leo平时学习用的绘本/教具
- [ ] 手机充满电 + 清理存储空间
- [ ] （可选）手机支架/三脚架（没有也行，手持拍即可）

**拍摄环境准备：**
- [ ] 找一个光线好的位置（靠窗最佳）
- [ ] 桌面清理干净，只放拍摄相关的物品
- [ ] 背景简洁（白色/木色桌面最好）
- [ ] 如果晚上拍，确保灯光明亮

### 5.2 必拍素材清单

```
⭐ 必拍（第一篇笔记必须用到的）：

□ 5个游戏的材料全家福
  → 把5个游戏的材料分别摆好，俯拍
  → 每个游戏拍2-3张不同角度

□ 关联记忆法（O→Q，P→R）
  → 拍爸爸在纸上画O变Q的过程
  → 最好拍过程（手在画）和成品（画好的）

□ 画动物游戏
  → 拍Leo认字母+画动物的过程
  → 拍完成的作品（字母+画的动物）

□ 抽字母组单词
  → 拍字母卡片散落在桌面的样子
  → 拍正在组单词的过程
  → 拍组成的单词成品

□ 亲子比赛
  → 拍爸爸和Leo面对面的背影/侧面
  → 拍比赛的过程

□ iHour数据截图
  → 截取最新的iHour数据页面
  → 包括总时长、各项占比饼图、每周投入柱状图
```

```
🌟 额外加分素材（如果有时间）：

□ 爸爸和Leo互动的温馨画面
  → 背影、手部特写、侧面
  → 自然状态，不用摆拍

□ Leo说英语/认字母的短视频
  → 15-30秒即可
  → 拍到声音最重要（"This is A!""B-Bear!"）
  → 画面可以只拍手/桌面/背影

□ Leo的表情/反应（自然抓拍）
  → 开心笑的时候
  → 答对了很兴奋的时候
  → 不用正面，侧面/背影都行

□ 日常启蒙场景
  → 早上磨耳朵时的场景
  → 绘本阅读时的场景
  → 看迪士尼美语时的场景

□ 爸爸画外音录制
  → 让爸爸对着手机讲解1-2个游戏
  → "嗨，这是我们家Leo最喜欢的字母游戏..."
  → 每段30秒-1分钟即可
  → （这个可以第一篇先不用，后续视频笔记用）
```

### 5.3 拍摄技巧速记

```
📸 拍照速记：

角度：俯拍为主（从上往下拍桌面）
光线：靠窗自然光最好，开灯也行但避免暖黄灯
构图：主体居中，周围留白
数量：每个场景多拍几张（至少3-5张），后面挑最好的
滤镜：先拍原图，后期统一调色

📹 拍视频速记：

时长：每段15-60秒
角度：俯拍桌面 或 侧面平拍
声音：保留原声（爸爸和Leo的对话）
稳定：尽量稳，可以靠在桌子上或椅子上
光线：同拍照
```

### 5.4 拍摄时间安排建议

```
周六拍摄时间线（约1.5-2小时）：

10:00-10:15  环境准备（整理桌面、调整光线）
10:15-10:30  拍摄游戏材料静物照
10:30-11:15  爸爸带Leo玩游戏，妈妈在旁边拍
             （按游戏1→2→3→4→5的顺序）
11:15-11:30  补拍一些日常互动花絮
11:30-11:45  （可选）录制爸爸画外音
11:45-12:00  检查素材是否够用

💡 关键原则：
让爸爸正常带Leo玩，你就在旁边拍。
不要为了拍摄打断他们的互动节奏。
最自然的画面 = 最好的素材！
```
