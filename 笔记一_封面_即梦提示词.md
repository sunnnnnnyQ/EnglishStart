## 即梦提示词公式说明

即梦的提示词结构：**主体 + 行为/状态 + 环境 + 风格 + 色彩 + 光影 + 构图**

即梦对中文非常友好，直接用中文写提示词即可，不需要英文。

### 方案A：童趣文字海报风（最推荐⭐）

```
一张小红书封面海报，竖版构图，3:4比例。

文字内容（必须准确生成以下文字）：
- 画面上半部分，居中显示大标题文字"字母启蒙从0到1"，使用圆润粗体手写风格字体，字号大且醒目，文字颜色为深棕色或深蓝色，每个字带有白色描边轮廓。标题下方一行小字"教学方法篇"。
- 大标题下方，居中显示副标题"34天认全26个字母的全过程"，字号比主标题小，颜色为天蓝色，手写体风格。
- 画面底部并排3个圆角标签小色块，从左到右分别写着："普通非英语专业爸妈""34天实录""手把手4个阶段"，标签底色分别为暖黄色、天蓝色、嫩绿色，文字为白色。

背景：干净的奶油白色/浅米色背景，局部有撕纸拼贴纹理装饰，增加手工质感。画面边角点缀少量童趣手绘小插画：一只小恐龙、几个彩色英文字母（A、B、C）、小星星、铅笔，作为氛围装饰但不抢主体文字。

风格：扁平插画风格结合手写文字排版，童趣可爱，色彩鲜明饱和。配色以暖黄、天蓝、嫩绿、奶油白为主。整体感觉温馨、活泼、有童真氛围，像一本儿童绘本的封面。

构图：文字占据画面主体（60%-70%面积），手绘插画装饰在四角和边缘，不遮挡文字。整体干净整洁，文字清晰易读。
```

## 🔧 即梦操作步骤

1. 打开即梦AI → 选择「图片生成」
2. 粘贴上面的提示词（推荐方案A）
3. 设置：
   - **尺寸**：选择 3:4（竖版）
   - **模型**：选择最新版本（4.0优先）
   - **生成数量**：一次生成4张，挑最好的
4. 生成后挑选最满意的一张

### ⚠️ 关于AI生成文字的说明

AI生成中文文字经常会出错（缺笔画、多字、乱码），这是目前所有AI绘图工具的通病。两种应对方式：

**方式一（推荐）：AI生成背景+手动加文字**
- 用提示词生成一张只有背景和装饰元素的图（去掉提示词中的文字内容描述）
- 然后用醒图/Canva/小红书编辑器手动叠加文字
- 文字内容：

```
主标题：字母启蒙从0到1·教学方法篇
副标题：34天认全26个字母的全过程

底部3个圆角标签：
  · 普通非英语专业爸妈
  · 34天实录
  · 手把手4个阶段
```

**方式二：直接用提示词让AI生成文字**
- 直接复制方案A的提示词，赌AI能正确生成文字
- 如果文字有小瑕疵，用醒图/PS修一下

## 💡 即梦提示词调优技巧

如果生成效果不理想，试试以下调整：

| 问题 | 解决方法 |
|------|----------|
| 画面太乱、元素太多 | 在提示词末尾加「简洁构图，元素不要太多，留白充足」 |
| 颜色不对 | 把「暖黄、天蓝」改成具体色号描述「明亮的橙黄色、清爽的天蓝色」 |
| 风格太写实 | 加强调「扁平矢量插画风格，非写实」 |
| 贴纸感不够 | 加「每个元素都带有白色粗描边和投影阴影，像真实贴纸贴在纸上」 |
| 出现了不想要的文字 | 加「画面中不要出现任何文字」 |
| 想要更童趣 | 加「幼儿园风格、儿童绘本插画感、圆润可爱的造型」 |

## 🎯 为什么这样写提示词

| 即梦规范要求 | 我的提示词如何对应 |
|-------------|-------------------|
| **主体要具体** | 明确列出了每个贴纸元素：字母、恐龙、铅笔、星星 |
| **避免抽象意境** | 没有写"童趣感""快乐氛围"，而是用具体的视觉元素来传达 |
| **指定用途** | 开头就说明「小红书封面海报」 |
| **风格明确** | 明确指定「扁平插画/手绘涂鸦/拼贴杂志」 |
| **色彩具体** | 列出了具体配色：暖黄、天蓝、嫩绿 |
| **构图说明** | 明确要求「中央留白、元素四周分布」 |
| **中文友好** | 全部使用中文，不需要英文翻译 |
