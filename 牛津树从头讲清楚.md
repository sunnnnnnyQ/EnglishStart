# 牛津树，从头讲清楚

---

## 一、牛津树是什么，怎么来的

### 1. 一句话说清

牛津树的全名叫 **Oxford Reading Tree**，是**英国牛津大学出版社**出的一套儿童分级阅读体系。

### 2. 它为什么会存在

英国小学需要一套东西来教孩子从零开始学会阅读。牛津大学出版社就做了这么一套书，从最简单的"一页一个词"一路走到"能自己读章节小说"。

所以它本质上是：

- **一套英国小学用的阅读教材**
- 不是绘本
- 不是课外读物
- 是**正式的阅读学习体系**

### 3. 它是怎么火的

- **1986 年**，核心系列 **Biff, Chip and Kipper Stories** 开始出版
- 作者是 **Roderick Hunt**，插画家是 **Alex Brychta**
- 因为故事围绕一家人展开，角色固定，孩子容易追，所以慢慢成了英国最受欢迎的分级读物
- 英国超过 **80% 的小学**选用牛津树

### 4. 它有多大

官方说法是 **800+ 本**，覆盖 **16 个级别**（Oxford Level 1 到 16），里面有十几个子系列。

所以牛津树不是一套书，而是**一棵很大的树，上面长了很多枝杈**。每条枝杈是一个子系列，各有各的侧重。

---

## 二、这棵树上到底有哪些枝杈（官方子系列）

你不需要全部记住。只需要知道它大致分成 **4 类功能**：

### A 类：故事主线

这是牛津树最核心的部分，也是大多数家长说的"刷牛津树"指的那部分。

| 子系列 | 说的是什么 |
|---|---|
| **Biff, Chip and Kipper Stories**（BCK 故事） | 一家人的日常故事，角色固定，故事连贯，L1-L9 |
| **Biff, Chip and Kipper Decode & Develop**（DD） | BCK 角色的进阶故事，拼读和故事融合，L1-L9 |
| **Traditional Tales** | 经典童话改编（三只小猪、金发姑娘等），L1-L9 |
| **Snapdragons** | 不是 BCK 一家人的独立故事，L1-L7 |
| **Story Sparks** | 成长主题故事，L1-L11 |
| **All Stars** | 高阶拓展故事，L9-L12 |

### B 类：自然拼读

这部分是帮孩子学"字母发什么音、怎么拼出来"的。

| 子系列 | 说的是什么 |
|---|---|
| **Floppy's Phonics Sounds & Letters** | 系统学字母音和拼读规则，L1-L6 |
| **Floppy's Phonics Fiction** | 用小故事来练拼读，L1-L6 |
| **Floppy's Phonics Non-fiction** | 用科普内容来练拼读，L1-L6 |
| **Songbirds Phonics** | Julia Donaldson 写的拼读练习故事，L1-L6 |

### C 类：科普 / 非虚构

这部分是拓展知识面的。

| 子系列 | 说的是什么 |
|---|---|
| **inFact** | 纪实小百科，L1-L11 |
| **Fireflies** | 非虚构百科，L1-L10 |
| **Explore with Biff, Chip and Kipper** | BCK 角色带你做趣味科普，虚构+非虚构配对，L1-L9 |

### D 类：家庭版

| 子系列 | 说的是什么 |
|---|---|
| **Read with Biff, Chip and Kipper** | 专门给家庭用的版本，含练习，L1-L6 |

### E 类：高阶

| 子系列 | 说的是什么 |
|---|---|
| **TreeTops** | 高阶章节书，L10-L16 |

## 三、国外原版和国内版本，到底是什么关系

### 1. 先说国外原版

原版就是牛津大学出版社自己出的英国版。上面那些子系列，全部加起来 800 多本，就是完整的原版。

### 2. 国内为什么会有各种版本

因为 800 多本太多了，而且原版价格很贵。所以国内出现了两条路：

- **正版引进**：有出版社花钱买了版权，挑选了其中一部分引进中国
- **祖国版**：没有版权，直接复刻印刷，价格便宜很多

### 3. 正版引进了哪些

国内先后有 **3 家** 做过正版引进，它们各自挑了不同的枝杈：

#### ① 凯迪克（已停运，版权转给了奇奇学）

- 引进了 **BCK 故事 + Floppy's Phonics**，L1-L9，大约 400 册
- 配 **毛毛虫点读笔**
- 还引进了 **家庭版**（Read with BCK），L1-L6，大约 58 册
- **现在已经买不到了**，凯迪克不再运营

#### ② 奇奇学（喜马拉雅旗下，2025 年起接手）

- 目前是 **BCK 系列中国大陆唯一正版**
- 引进了 L1-L9，大约 195 册
- 配 **专属 Floppy 点读笔**（注意：不兼容毛毛虫）
- 有配套课程、学情报告等增值服务

#### ③ 外研社（丽声系列）

外研社没有引进 BCK 故事主线，而是引进了**其他枝杈**，并且统一用了"丽声"这个品牌名：

| 外研社叫什么 | 其实是牛津树的什么 |
|---|---|
| 丽声拼读故事会 | Songbirds Phonics |
| 丽声妙想英文绘本 | Snapdragons |
| 丽声经典故事屋 | Traditional Tales |
| 丽声英语百科分级读物 | inFact |
| 丽声英语百科万花筒 | Fireflies |

配 **外研通点读笔**。

#### ④ 中国青年出版社（典范英语）

- 引进了 BCK 故事，但做了**重新编排**
- 原版一图一页，典范英语压缩成了四图一页
- **不支持点读**
- 教材感比较重，更像教辅而不是绘本

### 4. 一句话说清国内外区别

- **国外原版**：全系列 800+ 本，完整体系
- **国内正版引进**：各家挑了一部分，改了名字，配了不同的点读笔
- **祖国版**：没版权，复刻原版内容，便宜，质量参差不齐

## 四、祖国版的各种"标"，到底是什么

### 1. 先说清楚一件事

祖国版的"红标""无标""彩虹花""橙标"——这些名字**不是牛津官方的分类**。

它们是**不同印刷厂/渠道为了区分自家产品**，自己取的名字。

所以：

- 内容上，它们和原版**基本一样**
- 区别主要在：**印刷质量、点读功能、兼容的点读笔、有没有中文讲解**

我把公开资料里能找到的全部列出来。**粗略分成 3 档**：现在还能买到的、已经不好买的、已经淘汰的。

### 3. 所以各种标的区别到底是什么

**对，主要就是印刷质量、点读体验和配件赠品的区别，核心内容是一样的。**

不同的"标"就是不同印刷厂/渠道为了区分自家产品取的名字，类似同一款手机壳不同厂家做的。

更具体的选购建议：

- **你在意性价比、想买全级别** → **红标**
- **你在意印刷最像原版** → **无标**
- **你在意点读互动、家里有毛毛虫笔** → **彩虹花**
- **你想要最新版本、额外想看中国故事** → **新版四叶草**
- **你想要配件最丰富（闪卡等）** → **金标 / 钻石 / 未来之星**（但只有 L1-L2）
- **橙标** → 能买到但评价两极，不是首选
- **蓝标 / 黄标 / 绿标** → 已绝版，遇到二手再说

## 五、不同级别的牛津树，内容上有什么区别

### 1. 总览

牛津树用 **Oxford Level**（简称 OL）来分级，一共 16 级。家庭启蒙最常用的是 **L1 到 L9**。

| 级别 | 大致对应能力 | 内容长什么样 |
|---|---|---|
| **L1（无字书）** | 完全零基础 | 没有文字，只有图画，靠看图讲故事 |
| **L1+** | 刚开始接触英文 | 每页 1 个词或 1 个短句 |
| **L2** | 能听懂简单句 | 每页 1-2 个简单句，句型开始重复 |
| **L3** | 能跟读短句 | 句子稍长，开始有简单对话 |
| **L4** | 能理解简单故事 | 故事更完整，句式更丰富 |
| **L5** | 能跟着角色走故事 | 开始出现 magic key 奇幻冒险故事线 |
| **L6** | 能理解更复杂情节 | 故事更长，词汇量更大，开始有转折 |
| **L7-L9** | 能半独立阅读 | 接近桥梁书水平，故事完整度高 |
| **L10-L16（TreeTops）** | 独立阅读 | 章节书阶段，基本不需要家长陪 |

### 2. 内容上最核心的变化

从低到高，变化的不只是"字多了"，而是 **5 件事同时在变**：

- **句子长度**：从 1 个词 → 到一段话
- **句型复杂度**：从 "I like…" → 到 "If… then… because…"
- **故事复杂度**：从 "去公园" → 到 "穿越时空冒险"
- **词汇范围**：从日常高频词 → 到更低频的抽象词
- **图文比例**：从 90% 图 → 到 50% 图 50% 文字

### 3. 一个很重要的分水岭：L5

L5 开始，BCK 故事里出现了 **Magic Key**（魔法钥匙）。

从这里开始，故事从"一家人的日常生活"变成了"穿越冒险"。

很多孩子是在这个阶段被"勾住"的——因为故事开始有了悬念、有了奇幻元素。

但也有孩子在这里掉队——因为句子变长了、词汇变难了。

所以 L5 是一个关键节点：**如果孩子在 L4 之前没有积累够听力和兴趣，L5 会很容易卡住。**

## 六、建议怎么读

### 1. 不要从 L1 无字书开始

很多家长觉得应该从最低开始。但 L1 无字书对大多数 3 岁左右的孩子来说太简单了，而且没有文字输入，启蒙效率不高。

**建议直接从 L1+ 或 L2 开始**，除非孩子完全零基础、也没有任何英语输入经验。

### 2. 不要"刷完一级再升下一级"

牛津树同一级别有很多本（比如 L2 可能有几十本）。

不需要每本都读。

更好的方式是：

- 同一级别挑 **10-15 本** 读熟
- 然后往上走一级
- 如果发现上一级太难，回来再多读几本当前级别

### 3. 每本书怎么读

**第 1 遍**：你读给孩子听，孩子只看图

**第 2 遍**：你读，到重复句的地方停下来让孩子接

**第 3 遍**：用点读笔听，你在旁边

**第 4 遍以后**：孩子自己点读，或者只听音频

### 4. 不要只读不听

这是庆爸方法里最重要的一点：**读完之后一定要有"听"的环节**。

可以是：

- 点读笔反复点
- 配套音频放给孩子听
- 睡前当背景音听

目标是让孩子对同一批内容，从"看着图能懂"变成"不看图也能听懂"。

### 5. 不要一口气买整棵树

先买 **最低阶少量**（比如彩虹花 L1-L2），确认孩子喜欢牛津树的故事风格。

如果孩子喜欢 → 再继续买 L3-L5

如果孩子不太买账 → 说明他现阶段可能更适合其他类型的书（比如 FLR、故事绘本），牛津树后面再说

### 6. 读牛津树的同时不要丢掉其他线

牛津树很好，但它不是万能的。

它最强的是"故事主线 + 拼读配套"，但它的弱项是：

- 低阶阶段趣味性不如一些经典绘本
- 科普类不如 National Geographic Kids 或 DK 专业
- 主题多样性不如 FLR 或 I Can Read

所以最好的方式是：

- **牛津树做故事主线之一**
- **FLR / I Can Read 做可读性主线**
- **故事绘本做兴趣线**
- **科普小书做拓展线**

## 七、针对你家的建议

### 你家的情况

- 有毛毛虫和小蝌蚪点读笔
- 孩子 3 岁左右，词汇量约 200
- 已经有海尼曼、FLR、故事绘本
- 牛津树目前是"少量试水"的定位

### 我的建议

1. **先买彩虹花版 L1+ 和 L2**（大约二三十本就够试水）
   - 彩虹花是祖国版里唯一兼容毛毛虫的
   - 点读互动最丰富，低龄体验最好
2. **不要买 L1 无字书**
   - 对你家孩子来说太简单，没有文字输入
3. **不要同时开太多级别**
   - 先把 L2 读顺，再上 L3
4. **读完一定要听**
   - 用毛毛虫反复点
   - 或者放音频当背景
5. **如果孩子不喜欢，不要硬推**
   - 说明他目前更适合 FLR / 故事绘本这条线
   - 牛津树可以 3-6 个月后再试

### 一句话版

> **牛津树很好，但它只是你家英语启蒙路线里的一条线，不是全部。先用彩虹花 L1+/L2 少量试水，孩子喜欢再继续，不喜欢就先放一放。**
