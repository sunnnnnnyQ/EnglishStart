### 📍 阶段D：大循环可以启动了！（第10个月+ → 安安4岁+）

**当安安满足以下条件时，庆爸大循环方法可以正式启用：**

| 条件 | 预计达到时间 |
|------|------------|
| 年龄4岁+ | 约10个月后 |

| 能接受爸爸"读英文+解释"的模式 | 3岁可能不耐烦，4岁认知成熟 |

#### 大循环怎么启动

```
大循环操作（4岁+启动）：

① 爸爸每天1小时亲子共读牛津树
   → 爸爸读英文（指图）
   → 简单用中文解释不懂的地方（不用逐句翻译）
   → 再听一遍英文音频

② 从当前已读到的级别继续往上推
   → 如果常规路线已到Stage 4，直接从Stage 4开始大循环
   → 每读完一个阶段，不急着往上，先横向拓展：
     - 同级别的RAZ
     - 同级别的大猫
     - 同难度的绘本/桥梁书
   → 这就是"循环"：同一难度读透了再上升

③ 目标：大循环6-12个月内推到牛津树Stage 6-7
   → 到Stage 6后开始"裸听"训练
   → 裸听 = 不看书只听音频，能听懂70%以上
```

#### 大循环期间的素材

| 类别 | 素材 |
|------|------|
| 主线分级 | 牛津树 Stage 4-7 |
| 横向拓展 | RAZ E-J、大猫中级、培生基础-提高级 |
| 桥梁书 | Fly Guy、Nate the Great、Mercy Watson、Magic Tree House |
| 动画 | Little Fox 3-5级、Bluey、科普纪录片 |
| 裸听素材 | 牛津树音频、初章书音频 |
